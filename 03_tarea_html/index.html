Diccionario Maestro de HTML

Introducción al lenguaje

HTML es el esqueleto de una web que utiliza etiquetas para organizar el contenido; estas le indican al navegador qué parte es un título, qué es un párrafo o dónde hay un enlace, estableciendo la estructura esencial sin preocuparse por el diseño o la interactividad. es el orden y lo que le da sentido al sitio web, sin HTML seria desordenado y careceria de sentido.
Ayuda a que el navegador sepa cómo mostrarlo al usuario.

Etiquetas de Formato

La etiqueta `br` sirve para insertar un salto de línea en HTML. Permite bajar a la siguiente línea de texto sin crear un nuevo párrafo ni añadir espacio adicional entre líneas.
En HTML, la etiqueta "hr" significa Horizontal Rule (regla horizontal). Se utiliza para crear una línea divisoria que separa secciones de contenido, indicando un cambio de tema o una pausa temática en la página.
